# SVIMOC-Net
SVIMOC-Net: Semi-Supervised Learning for Video Indiscernible Marine Object Counting with First-Frame-Only Annotation

## Setup
Pytorch 1.10.2

Python 3.8.19

## Dataset preparation 
We construct a new dataset, termed VIMOC-1F, which contains 50 underwater video sequences with annotations provided only on the first frame (5,776 points in total).
